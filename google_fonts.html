Google Fonts :

Please choose a font type.

Step 1 : Link to google fonts.

Each font has to be retrieved from the google cloud server. Copy the link from the codebox below and add it to the head section of your html.

<link href="https://fonts.googleapis.com/css2?[ADD FONTS HERE]&display=swap" rel="stylesheet">

Step 2 : Font Family Identifier

Each font has a unique identifier such as (&family=serif). Just change the identifer in the command line then add the new font to your styles. The style rule is in step 3.

<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

Step 3 : Add weight to your font.

Choose a font weight from the list and the codebox will reflect the changes. Copy and paste the code to your stylesheet rules.

<link href="https://fonts.googleapis.com/css2?family=Roboto&family=Playfair+Display&display=swap" rel="stylesheet">
